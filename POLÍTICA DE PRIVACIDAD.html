Política de Privacidad de WrapCalc Última actualización: [08/06/2025] La aplicación WrapCalc no recopila, almacena ni comparte ningún tipo de información personal del usuario. Uso de WebView La aplicación accede a un sitio web mediante una vista integrada (WebView), sin recopilar cookies, datos personales ni realizar seguimiento del usuario. Permisos La única funcionalidad usada es el acceso a Internet, necesario para cargar el contenido desde el sitio web. Contacto Si tienes dudas sobre esta política, puedes contactar con el desarrollador a través del correo: [calcwrap@proton.me].
