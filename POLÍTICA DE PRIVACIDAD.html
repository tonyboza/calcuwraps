Política de Privacidad

WrapCalc no recopila, almacena ni comparte ningún dato personal del usuario.

La aplicación no requiere registro ni inicio de sesión.

WrapCalc solo utiliza conexión a Internet para cargar el contenido web actualizado desde GitHub Pages, si está disponible. En modo offline, sigue funcionando sin restricciones.

No utilizamos cookies, análisis de terceros, ni recopilamos ubicaciones ni contactos.

Si tienes dudas, puedes contactar con el desarrollador en: calcwrapoton.me
